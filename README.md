# DevForge

DevForge is an AI-powered developer workspace with GitHub integration, AI tooling, and project/history management.

## Monorepo Structure

- `frontend/` React + TypeScript + Tailwind + shadcn-style component primitives
- `backend/` Node.js + Express API service
- `database/prisma/` Prisma schema for PostgreSQL

## Quick Start

1. Install dependencies in both apps.
2. Configure `.env` files from the provided examples.
3. Run Prisma migrations.
4. Start backend and frontend dev servers.
